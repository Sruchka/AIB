Справочник "Внешние учетные записи"

Справочник содержит перечень учетных записей для подключения к ИБ, Серверам 1С, СУБД SQL, Хранилищу конфигураций. Тип учетной записи определяет назначение и дальнейшее использование учетной записи в системе. Возможна настройка подключения по схеме Имя/Пароль или с использованием доменной авторизации. Для типа учетной записи "Учетная запись ИБ" (т.е. используемая для подключения к базам 1С) параметр "Код блокировки сеансов" содержит пароль устанавливаемый на блокировку сеансов ИБ (параметр /UC пакетного запуска). Автоматическая установка блокировки сеансов ИБ выполняется при запуске процедуры установки обновлений. Если код блокировки не задан, то автоматическая блокировка сеансов не устанавливается.
